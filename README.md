## :octocat: Setup

### Installation

```sh
# Clone this repo
git@github.com:filipe1309/ctv-news-api.git

# Enter project folder
cd ctv-news-api
```

### 🏃 Runnning

```sh
./bin/runenv
```

The app will run at: http://localhost:5001
